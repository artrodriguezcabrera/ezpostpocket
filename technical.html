ez Post Pocket

Designed and tested with a minimum 2" x 8" wood sub structure, the Post Pocket's innovative and patented design delivers the quickest, easiest and most economical way to install a code compliant 4 x 4 wood guard rail post for your railing systems.

This bracket integrates a 2" x 8" support block that allows your wood post to securely, attach directly to the decks substructure, tying together and reinforcing all the supporting wood joists for added strength while allowing the post to be perfectly plumbed at installation.

KEY FEATURES

  • Quickly and easily installs to a minimum 2" x 8" wood sub structure
  • Wood post can be plumbed with provided shims
  • Built in pressure tabs insure a tight post fit
  • Weep hole provided for water drainage
  • Protective corrosion resistance coating
  • Strong and secure structural design
  • Mounting hardware Included
  • Fully tested and code compliant
